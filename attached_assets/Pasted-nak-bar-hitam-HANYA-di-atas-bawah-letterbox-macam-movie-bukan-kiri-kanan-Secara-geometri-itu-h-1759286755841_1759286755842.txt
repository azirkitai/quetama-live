Matlamat: Utamakan bar hitam atas/bawah. Jika skrin terlalu sempit (lebih sempit dari 16:9), automatik fallback ke mode biasa (pillarbox kiri/kanan) supaya tak terpotong.

1) Pastikan wrapper center penuh skrin
// bila isFullscreen === true
return (
  <div className="tv-viewport fixed inset-0 grid place-items-center bg-black overflow-hidden">
    <div ref={stageRef} id="stage" className="grid" style={stageStyle}>
      {renderContent()}
    </div>
  </div>
);

2) Ganti fungsi autoscale (fitStage)
useEffect(() => {
  if (!isFullscreen || !stageRef.current) return;

  const W = 1920, H = 1080;
  const stage = stageRef.current;
  const viewport = stage.parentElement!; // .tv-viewport

  function fitStage() {
    const vw = viewport.clientWidth;
    const vh = viewport.clientHeight;

    // CALC: skala ikut LEBAR (untuk letterbox atas/bawah)
    const scaleWidthFit = vw / W;
    const scaledHeight  = H * scaleWidthFit;

    let scale: number;

    if (scaledHeight <= vh) {
      // âœ… Muat tinggi â†’ bar hitam akan muncul ATAS/BAWAH sahaja
      scale = scaleWidthFit;
    } else {
      // â— Skrin terlalu sempit untuk letterbox atas/bawah tanpa crop
      //    â†’ fallback supaya tak terpotong (pillarbox kiri/kanan)
      const scaleContain = Math.min(vw / W, vh / H);
      scale = scaleContain;
    }

    stage.style.transformOrigin = 'center center';
    stage.style.transform = `scale(${scale})`;
  }

  fitStage();
  window.addEventListener('resize', fitStage);
  window.addEventListener('orientationchange', fitStage);
  return () => {
    window.removeEventListener('resize', fitStage);
    window.removeEventListener('orientationchange', fitStage);
  };
}, [isFullscreen]);


Kalau kau memang nak paksa atas/bawah sahaja walaupun skrin sempit (akan crop), tukar bahagian else kepada scale = scaleWidthFit; dan pastikan #stage parent overflow:hidden.
