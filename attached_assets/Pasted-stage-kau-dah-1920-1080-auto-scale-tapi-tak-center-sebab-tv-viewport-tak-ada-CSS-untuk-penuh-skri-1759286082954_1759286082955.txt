3) Kekalkan JS auto-scale (yang kau dah ada)

Kod kau ni dah betulâ€”gunakan parent (tv-viewport) sebagai viewport:

useEffect(() => {
  if (!isFullscreen || !stageRef.current) return;
  const STAGE_WIDTH = 1920, STAGE_HEIGHT = 1080;
  const stage = stageRef.current;
  const viewport = stage.parentElement!;
  const fitStage = () => {
    const s = Math.min(viewport.clientWidth / STAGE_WIDTH, viewport.clientHeight / STAGE_HEIGHT);
    stage.style.transform = `scale(${s})`;        // center dijaga oleh grid
    stage.style.transformOrigin = 'center center';
  };
  fitStage();
  addEventListener('resize', fitStage);
  addEventListener('orientationchange', fitStage);
  return () => {
    removeEventListener('resize', fitStage);
    removeEventListener('orientationchange', fitStage);
  };
}, [isFullscreen]);


Penting: jangan letak position:fixed / translate(-50%,-50%) pada #stage. Pusatkan melalui parent .tv-viewport (grid) sahaja.
